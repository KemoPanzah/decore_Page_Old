decore Base | UI fastly#

Introduction#

decore Base is an out-of-the-box “Python to Vue.js” data application dashboard that helps you go from idea to view in a few simple steps. It is aimed at those who want to focus on the results of their algorithms, do scientific work, or perform teaching and learning functions.

Functions and integrations#

  • Finished SPA (Single Page Application) with Vue.js using the Quasar Framework (quasarframework/quasar)

  • Predefined webapi for metadata and actions with Flask (pallets/flask)

  • Integrated ORM for data management and database interfaces (SQLite) powered by the great Peewee (coleifer/peewee)

  • Data verification with Cerberus (pyeve/cerberus)

  • Password management with pykeepass (libkeepass/pykeepass)

Please support these great projects!
